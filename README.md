# Organizador de Arquivos

Projeto de automação desenvolvido em Python com o objetivo de organizar arquivos automaticamente em subpastas de acordo com suas extensões.  
Além da organização, o sistema também gera um relatório final com os arquivos movimentados.

## Funcionalidades
- leitura dos arquivos de uma pasta
- identificação de extensões
- criação automática de subpastas por categoria
- movimentação de arquivos
- geração de relatório em arquivo `.txt`

## Tecnologias utilizadas
- Python

## Objetivo do projeto
Este projeto foi desenvolvido para praticar automação de tarefas com Python, manipulação de arquivos e diretórios, além de organização lógica de processos.

## Como executar
1. Certifique-se de ter o Python instalado
2. Baixe ou clone este repositório
3. Abra o terminal na pasta do projeto
4. Execute o comando:

```bash
python organizador_arquivos.py
```
